-Para Jogar você deve ter instalado a biblioteca pygame.
-O executável do jogo é o arquivo menu.

Regras Do Jogo:
-Após a seleção de personagens o jogo acaba após um jogador ser declarado como vencedor.
-O Ki se recupera em 25% a cada 5 segundos.
-O dano só se aplica quando há ki.

As instruções de comandos são:

Player 1: 
Usa as teclas: 
'W' Para mover para cima
'A' Para mover-se para Esquerda.
'D' Para mover-se para a Direita'
'S' para mover-se para baixo.
'J' para ataques  normais.
'K' para ataques especiais

Player 2: 
Usa as teclas: 
'⬆️' (Seta para Cima) Para mover para cima
'⬅️' (Seta para esquerda) Para mover-se para Esquerda.
'➡️'(Seta para direita) Para mover-se para a Direita'
'⬇️' (Seta para baixo) para mover-se para baixo.
'0' para ataques  normais.
'1' para ataques especiais

-To play you must have the pygame library installed.
-The game's executable is the menu file.

Game Rules:
-After character selection, the game ends after one player is declared the winner.
-Ki recovers by 25% every 5 seconds.
-Damage only applies when there is ki.

The command instructions are:

Player 1: 
Use the keys: 
'W' To move up
'A' To move Left.
'D' To move Right'
'S' to move down.
'J' for normal attacks.
'K' for special attacks

Player 2: 
Use the keys: 
'⬆️' (Up Arrow) To move up
'⬅️' (Left Arrow) To move Left.
'➡️'(Right Arrow) To move Right'
'⬇️' (Down arrow) to move down.
'0' for normal attacks.
'1' for special attacks
